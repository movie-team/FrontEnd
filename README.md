## 깃 명령어
브랜치 생성 : git branch 브랜치이름
대표 브랜치 변경 : git checkout 브랜치이름
로컬 브랜치 목록 조회 : git branch
원격 브랜치 목록 조회 : git branch -r
모든 브랜치 목록 조회 : git branch -a
